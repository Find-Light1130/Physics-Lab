# 【ZH-CN】

## 物理实验模拟器 - 开发蓝图与拓展计划

本项目基于 HTML5 Canvas、Tailwind CSS 和原生 JavaScript 构建。为了应对后续可能增加的新实验，核心架构已完全采用 **App 模块化路由**，未来新增实验只需在 `experiments.js` 中追加新模块即可。

### 一、 当前已完成的核心优化 (UI/UX 与渲染)

目前的代码版本已完成了以下深度优化：

1.  **彻底修复图表坐标轴偏移**：重写了 `drawLineChart` 的 Y 轴缩放逻辑。无论数据初值多小，中心零线（虚线 X 轴）都会**绝对居中**于画布中心，上下留白对称，解决了单摆初态图像被压在底部的痛点。
2.  **统一物理物体视觉风格**：单摆的摆球、自由落体的小球、匀变速的金属滑块，现在均统一采用银灰色 `#f1f5f9` 到 `#64748b` 的金属质感径向渐变，并带有符合物理规律的投影（悬浮/落地光影）。
3.  **自由落体虚影修正**：去除了原版蓝色的虚影，改为与主球一致的**半透明银灰色金属质感虚影**，视觉上完美融合。
4.  **速度矢量箭头化**：单摆和滑块的速度指示，已由简单的短横线或纯线条，升级为带有明确张角的**黄色实心矢量箭头**。
5.  **参数输入框美化**：去除了输入框外层的深色“胶囊背景”，改为透明底。数值强制**靠左对齐**，单位强制**靠右对齐**，排版更加清爽。
6.  **消除双分割线**：修复了实时数据面板由于底边框和容器分割冲突导致的“两条重叠横线”问题，现精准保留一条细线。
7.  **防抖与快速切换修复**：加入高频请求的节流与防抖机制。无论用户多快点击右上角的“昼夜切换”按钮，界面文字及 KaTeX 公式都不会因渲染冲突而消失。
8.  **本地文件绝对兼容**：彻底兼容了直接双击 `index.html` 在 `file:///` 协议下的运行。自动降级使用 `#` 路由，彻底避免了浏览器对 History API 的拦截导致的 404 与白屏错误。

### 二、 下一阶段 UI 交互增强建议

待平台基本成型后，建议在界面交互上继续添加以下细节，提升专业感：

1.  **图表坐标轴标签**：在 v-t 和 s-t 图像的下方和左侧，添加如 `t/s`, `v/(m·s⁻¹)`, `s/m` 的坐标轴标签。这将会让图表看起来符合标准教材插图的规范。
2.  **物理临界点反馈动画**：当自由落体小球接触地面、滑块到达斜面极限时，增加轻微的**物理回弹缩放动画**或**呼吸闪烁特效**。增强交互的“真实感”。
3.  **画布 (Canvas) 自适应缩放**：目前固定为 `800x450`。后续可通过 `ResizeObserver` 或监听 `window.resize`，让画布按窗口比例自适应缩放，确保在宽屏电脑和窄屏手机上均能完整显示。

### 三、 未来可拓展实验库 (设计蓝图)

基于现成的模块化 `App.registerModule` 机制，后续可以无缝加入以下实验模块：

#### 1. [二维运动] 平抛运动实验
*   **物理逻辑**：水平方向匀速直线运动 `x = v_x * t`，竖直方向自由落体 `y = 0.5 * g * t^2`。
*   **可视化重点**：发射一颗银灰色小球，在空中绘制出**半透明的抛物线轨迹**。数据面板分别展示水平速度、竖直速度和合速度的大小；画布上同时展示两个分速度的独立矢量箭头。
*   **教学意义**：直观演示运动的合成与分解。

#### 2. [非线性动力] 双摆混沌实验
*   **物理逻辑**：求解双摆的拉格朗日方程。两个摆首尾相连，对初始角度极其敏感（蝴蝶效应）。
*   **可视化重点**：下方摆锤会拖拽出一条**发光渐变的长轨迹线**。随着时间推移，画布上将绘制出极其复杂、永不重复、极具艺术美感的混沌对称图形。
*   **教学意义**：生动展示非线性动力学系统的混沌现象。

#### 3. [天体物理] 行星轨道与万有引力
*   **物理逻辑**：二体引力模拟 (核心公式 \(F = G \frac{Mm}{r^2}\) 的矢量积分)。
*   **可视化重点**：画布中央放置一个发光发热的金色“恒星”，围绕其运行一个带有水平初速度的“行星”。行星自带**发光的椭圆轨道拖尾**。行星上同时用不同颜色指示着**引力方向**和**运动方向**的矢量箭头。通过滑块调节行星初速度（可观测到不同的椭圆/抛物线轨道）。
*   **教学意义**：验证开普勒三大定律和牛顿万有引力定律。

#### 4. [精密实验] 卡文迪许扭秤
*   **物理逻辑**：光杠杆放大原理。利用微小的万有引力矩，通过镜面反射，在长距离刻度尺上放大成可见的位移。
*   **可视化重点**：画布模拟一个暗室结构。左右两巨大铅球，中间一根扭丝悬挂小铅球。旁边有一个**激光发射器**。拖拽大铅球靠近时，激光光斑在右侧标尺上会产生**肉眼可见的平滑位移**。
*   **教学意义**：展示精密物理实验的设计思想。

#### 5. [现代物理] 时空弯曲与引力透镜
*   **物理逻辑**：质量扭曲周围空间网格。
*   **可视化重点**：绘制大量的十字交叉背景网格线。当用户通过滑块召唤出一个大质量“天体”时，**网格线会像被拉伸的弹簧床垫一样陷入弯曲**。紧接着从画布左侧发射一束绿色的光，光在经过弯曲的网格时将发生**明显的偏折**。
*   **教学意义**：用极简的图像解释爱因斯坦广义相对论中的“引力是时空弯曲的表现”。

#### 6. [力学] 托里拆利实验
物理逻辑：将一端封闭的玻璃管灌满水银后倒扣在水银槽中，管中水银柱下降至一定高度后保持稳定。该水银柱的压强等于外界大气压强。

可视化重点：水银柱上方为“托里拆利真空”的空间，以及水银柱高度随海拔变化的动态示意图。

教学意义：首次制造了真空，并定量测量了大气压的大小，是流体静力学和气压研究的基石。

#### 7. [力学] 马德堡半球实验
物理逻辑：将两个空心铜半球对接，抽出内部空气，外部大气压的巨大压力使两个半球紧密贴合，需用多匹马才能拉开。

可视化重点：两队马匹分别向两侧奋力拉扯铜半球的壮观场景图。

教学意义：以极具戏剧性和视觉冲击力的方式，向大众（甚至包括当时的贵族）证明了大气压不仅存在，而且力量巨大。

热力学与分子运动论
#### 8. [热力学] 热功当量实验
物理逻辑：通过多种方式（如叶轮搅拌水、摩擦生热）对水做功，精确测量水温升高所需的功与热量的比值，证明机械功与热量可以相互转化。

可视化重点：焦耳实验装置的剖面图，显示重物下落带动水中叶轮旋转。

教学意义：为能量守恒定律（热力学第一定律）提供了最坚实的实验基础，统一了“功”与“热”两个看似无关的物理概念。

#### 9. [统计力学] 布朗运动实验
物理逻辑：在显微镜下观察悬浮在水中的花粉微粒，发现它们永不停息地进行无规则运动。这种运动源于水分子热运动对微粒撞击的不平衡。

可视化重点：显微镜下花粉微粒随机折线运动的动态轨迹图。

教学意义：为分子原子论和统计物理提供了关键的微观图像证据，是连接宏观世界与微观世界的桥梁。

电磁学
#### 10. [电磁学] 奥斯特实验
物理逻辑：将通电导线平行放置于小磁针上方，发现磁针发生偏转。表明电流（运动电荷）在其周围会产生磁场。

可视化重点：导线与磁针的相对位置图，以及通电前后磁针偏转的对比。

教学意义：揭示了电与磁之间并非孤立，而是存在深刻的内在联系，开启了电磁学统一的序幕。

#### 11. [电磁学] 电磁感应实验
物理逻辑：当穿过闭合线圈的磁通量发生变化时（如磁铁插入或拔出线圈），线圈中会产生感应电流。

可视化重点：磁铁在线圈中运动，同时连接线圈的电流计指针发生偏转的动态图。

教学意义：揭示了变化的磁场可以产生电场，是发电机和现代电力工业的根本原理。

#### 12. [电磁学] 赫兹实验
物理逻辑：利用振荡电路产生电磁波，并用谐振接收器探测到该波，证实了电磁波在真空中以光速传播，且具有反射、折射等波动特性。

可视化重点：赫兹火花发射器和接收环的装置图，以及接收环间隙中产生火花的细节。

教学意义：通过实验完美验证了麦克斯韦电磁场理论的预言，使电磁波从方程走向现实应用。

光学
#### 13. [光学] 棱镜色散实验
物理逻辑：让一束白光通过三棱镜，不同颜色的光因折射率不同而被分散开来，形成彩色光谱。再将光谱通过第二个棱镜可复合为白光。

可视化重点：三棱镜将一束白光展开为七色光谱带的经典图像。

教学意义：揭示了白光的复合本质，打破了“颜色是物体本身属性”的旧观念，是光谱学的开端。

#### 14. [光学] 杨氏双缝干涉实验
物理逻辑：将一束相干光通过两个相距很近的狭缝，两列光波在空间相遇叠加，在光屏上形成明暗相间的干涉条纹。

可视化重点：光屏上清晰、等间距的明暗条纹图案，以及说明波峰与波峰相遇加强、波峰与波谷相遇抵消的示意图。

教学意义：以极其简洁明确的实验证据，强有力地证明了光的波动性，终结了牛顿微粒说的长期统治。

近代物理学
#### 15. [原子物理] 油滴实验
物理逻辑：在电场中观察并测量带电油滴在重力和电场力平衡时悬浮的状态，计算出油滴所带电荷量。发现所有电荷量都是某个最小值的整数倍。

可视化重点：显微镜视野中一个悬浮的油滴，以及电场力与重力平衡的受力分析图。

教学意义：首次直接、精确地测定了基本电荷（电子电荷）的值，证明了电荷的量子化特性。

#### 16. [原子核物理] α粒子散射实验
物理逻辑：用放射性元素发出的α粒子束轰击极薄的金箔，发现绝大多数α粒子穿过，少数发生大角度偏转，极个别被弹回。

可视化重点：α粒子穿过金箔后在不同方向（特别是大角度）散射的轨迹分布图。

教学意义：彻底否定了“西瓜式”原子模型，确立了原子的“核式结构”模型（即原子内大部分是空的，中心有一个带正电的原子核）。

17. [量子力学] 电子双缝干涉实验
物理逻辑：将杨氏双缝实验中的光源替换为电子枪，即使让电子一个一个地发射，在底片上最终仍会形成干涉条纹。

可视化重点：底片上逐渐积累的电子点阵，最终形成明暗条纹的过程图。

教学意义：这是量子力学的核心实验，它无可辩驳地展示了微观粒子的波粒二象性，颠覆了经典的宏观认知。

18. [量子力学] 光电效应实验
物理逻辑：用不同频率的光照射金属表面，只有当光频率高于某一阈值时才会逸出光电子，且光电子能量与光强无关，只与频率有关。

可视化重点：实验装置图，以及光电流与电压、频率关系的特征曲线。

教学意义：该实验现象无法用经典波动理论解释，直接催生了爱因斯坦的光量子理论，是量子力学诞生的关键实验基础。

19. [量子力学] 康普顿散射实验
物理逻辑：用X射线照射石墨，发现散射后的X射线波长变长，且波长变化量随散射角变化。这证明了光子与电子碰撞时遵循能量和动量守恒。

可视化重点：不同散射角度下，原始X射线峰和波长变长的散射峰的对比图谱。

教学意义：为光的粒子性提供了最直接、最确凿的实验证据，使光的波粒二象性成为无可争议的事实。

前沿探索实验
20. [宇宙学/相对论] 发现引力波
物理逻辑：由两个致密天体（如黑洞、中子星）并合产生的剧烈时空扰动，以波的形式在宇宙中传播。LIGO利用激光干涉仪测量这种极其微小的空间拉伸与压缩。

可视化重点：LIGO长达4公里的L型真空管道示意图，以及探测到引力波信号后转换为声波的波形图。



### 四、 模块化技术栈总结

*   **路由技术**：利用 `window.location.hash` 配合 `hashchange` 事件，实现单页应用 (SPA) 的页面无刷新切换，完美兼容本地 `file://` 环境。
*   **物理引擎**：采用显式欧拉积分处理帧时间 `dt`，配合 `requestAnimationFrame` 实现最高 60fps 的流畅物理模拟。
*   **图形绘制**：基于原生 Canvas API，使用 **贝塞尔曲线 (`bezierCurveTo`)** 实现极度平滑的图表和运动轨迹重绘。
*   **UI 框架**：Tailwind CSS 搭建框架，配合自定义毛玻璃 CSS 变量，实现全天候无感的明暗模式自适应。 KaTeX 实现数学公式的动态渲染。
